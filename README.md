This project shows how to save and read data in Java using:
1. Text file (file I/O) for Drug data
2. PostgreSQL database (JDBC) for Patient data

Menu options:
1) Save drug to file
2) Show all drugs from file
3) Save patient to database
4) Show all patients from database
0) Exit

Files:
- Drugs are stored in drugs.txt (auto-created)

Database setup:
- PostgreSQL must be running
- Create a database and a table named "patients"
- Update DB_URL, DB_USER, DB_PASS in Main.java
- Add PostgreSQL JDBC driver

Notes:
- If the database is not set up, file features still work
- Input is validated to prevent crashes
